Manipulação de Dados

Para guardar um dado em uma variavel utiliza-se

EX:


    var nome = window.prompt('Qual é o seu nome ?')
    window.alert('É um grande prazer te conhecer,'+ nome)

Para unir os dados utiliza-se "+" Concatenação. Como mostrado no codigo acima

Conversão String > Número

Para mostrar ao Javascript que o "+" não é de concatenação precisa fazer a conversão de String para Número e usado o seguinte comando:


                    
     Number.parseInt (n)
     Number.parseFloat(n)
     Number(n)

Se não, ele vai entender que é concatenado como no codigo abaixo, ao invés de somar as variaveis, ele vai colocar os dois números dados um do lado do outro.



     var n1 = window.prompt('Digite um número:')

     var n2 = window.prompt ('Digite outro número')

     var s = n1 + n2 

     window.alert('A soma dos valores é' + s)

Se colocar no primeiro (1) e no segundo (1) o resultado será (11) na forma como o código está acima.

Para resolver isso usa-se o seguinte comando:

        

      var n1 = Number.parseInt(window.prompt('Digite um número:'))
      var n2 = Number.parseInt( window.prompt ('Digite outro número'))
      var s = n1 + n2 
      window.alert('A soma dos valores é  ' + s)

Dessa forma o resultado das variaveis será somado.
