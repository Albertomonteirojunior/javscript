Formatação de string

Aprendendo 3 formatações simples de string:

Tipos de Formatação

  • s.lengh - vê o tamanho da string
  • s.toUpperCase()- Deixa a string com caixa auta
  • s.toLowerCase() - Deixa a string com minusculas

EX:

   
       var nome = window.prompt('Qual é seu nome ?')
       Document.write(`Olá,${nome}! Seu nome tem ${nome.length}`)
       document.write(`Seu nome em maiúsculas é ${nome.toUpperCase()}`)
       document.write(`Seu nome em miúsculas é ${nome.toLowerCase()}`)

Formatando Números

Criando uma Variavel n1 = 1545.5 para deixar o valor com casa decimal usa-se:

        
      >var n1 = 1545.5
      >n1.toFixed(2)
      >'1545.50'

Para trocar o ponto por virgula fica assim:

        
      >var n1 = 1545.5
      >n1.toFixed(2).replace('.',',')
      >'1545,50'

Como mostra esse valor em real

        
       >n1.toLocaleString('pt-br',{ style: 'currency',currency:'BRL'}) 
       >R$ 1,545.50

Tipos de Formatação

  • toFixed (2) - para mostrar o numero com casa decimal
  • toFixed().replace('.',',')- Para trocar o ponto por virgula
  • .LocaleString('pt-BR',{style:'currency',currency:'BRL'})
