Criando um alert

Para criar um "alert" usa-se esse seguinte código:

       
           script

    window.alert('Minha primeira mensagem')

           /script

Para criar uma confirmação:

    
        script

     window.confirm('Está gostando de JS')
       
       /script

para chamar o prompt:

      
          script

    window.prompt('Qual é o seu nome ?')

          /script
