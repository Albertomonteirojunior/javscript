Oque são variaveis

Variaveis são espaços reservados na memória para guardar dados.

Utiliza-se nomes, numeros ou letras para identificar uma variavel, como na imagem abaixo:

Para guardar um valor em uma variavel em Javscript utiliza esse sinal " = " que sozinho se traduz recebe. Logo:


                Vaga a1 = carro1

Se lê : vaga a1 recebe carro1.

Exemplo de variavel:


               > var nome ="Gustavo"
               > undefined
               > nomes
               >'Gustavo'

Identificadores:

  • Podem começar com Letra, $ ou _
  • Não podem começar com números
  • É possivel usar letras e números
  • É possivel usar letras e números
  • É possivel usar acentos e símbolos
  • Não pode conter espaços
  • Não podem ser palavras resevadas

Dica:

  • Maiúsculas e minúsculas
    fazem diferença
  • tente escolher nomes
    coerentes para as variaveis

Tipos primitivos

o Tipo primitivo Number guarda numeros sendo eles inteiros ou reais, String guarda letras pontos e traços, e a Boolean valores como True e False.

Para saber qual é o tipo, tem um comando em Javscript usando a ferramenta NodeJs 


            >var n = 200
            >n
            >200
            >typeof n 
            >'number'
